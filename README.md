# Windows Forensics Analyzer (NX212)

## Overview
An automated forensic tool for HDD and memory analysis. Created during the NX212 unit to expedite the extraction of digital artifacts and initial forensic evidence assessment.

## Key Features
* **Automated Toolchain:** Installs and configures necessary forensic tools (Volatility, Bulk-Extractor, Binwalk).
* **Artifact Extraction:** Scans evidence files for traces and hidden data automatically.
* **Memory Analysis:** Identifies memory profiles and extracts network/process data using Volatility.
* **Forensic Report:** Generates a comprehensive summary of findings and identified artifacts.

## Ecosystem Impact
Serves as an accessible automation framework for students starting their journey in digital forensics and incident response.
